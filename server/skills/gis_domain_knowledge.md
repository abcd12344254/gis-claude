## GIS 领域知识

### 坐标系与投影

- **地理坐标系 (GCS)**：使用经纬度表示位置（EPSG:4326 = WGS84）
- **投影坐标系 (PCS)**：将球面投影到平面，单位通常为米
- **中国常用坐标系**：WGS84 (GPS) / GCJ-02 (高德国测局) / BD-09 (百度) / CGCS2000 (国家大地坐标系)
- **GCJ-02 与 WGS84 偏移**：300-500 米，不可混用
- **Web 墨卡托 (EPSG:3857)**：Web 地图标准投影，等角但面积变形大

### 空间分析概念

| 分析类型 | 概念 | 典型应用 |
|----------|------|----------|
| 缓冲区 | 围绕要素创建指定距离的区域 | 服务区、影响范围 |
| 叠加分析 | 两个图层空间关系的计算（交/并/差） | 用地分析、选址 |
| 空间聚类 | 基于距离密度将点分组 | 犯罪热点、客户分布 |
| 空间插值 | 从已知点推算未知点的值 | DEM生成、环境监测 |
| 网络分析 | 基于路网的路径和可达性计算 | 路径规划、设施选址 |
| 热点分析 | 空间自相关统计显著性检验 | 疾病分布、事故高发区 |
| 适宜性评价 | 多因子加权叠加评估 | 土地适宜性、选址 |

### 数据格式

| 格式 | 类型 | 适用场景 |
|------|------|----------|
| Shapefile (.shp) | 矢量 | 传统 GIS 数据交换 |
| GeoJSON | 矢量 | Web 地图、API 交互 |
| GeoPackage (.gpkg) | 矢量/栅格 | 现代替代 Shapefile |
| GeoTIFF (.tif) | 栅格 | DEM、卫星影像 |
| KML/KMZ | 矢量 | Google Earth |
| CSV (含坐标) | 表格 | 点数据导入 |

### 最佳实践

1. **面积计算** → 必须投影坐标系（EPSG:4326 计算面积会严重偏小）
2. **距离测量** → 必须投影坐标系或使用球面距离算法
3. **多源数据融合** → 统一到同一坐标系后再分析
4. **中国数据** → 注意 GCJ-02/WGS-84 转换
5. **大数据量** → 空间索引 + 分块处理
6. **几何质量** → 分析前检查几何有效性
