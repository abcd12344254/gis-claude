## 参数隐患与智能处理

### 缓冲区分析参数

| 参数隐患 | 问题描述 | 智能处理 |
|----------|----------|----------|
| **距离单位不明确** | 用户说"缓冲1公里"，但坐标系单位为度 | 自动检测 CRS 单位，若为度则提示需先转换 |
| **缓冲距离过大/过小** | 距离值相对于数据范围不合理 | 基于数据边界计算合理缓冲区间 |
| **缓冲后拓扑错误** | 自相交或无效几何 | 自动调用 fix_invalid_geometry |
| **多部件缓冲区** | 相邻缓冲区是否需要合并 | 默认不合并，根据用户需求决定 |

### 叠加分析参数

| 参数隐患 | 问题描述 | 智能处理 |
|----------|----------|----------|
| **几何类型不兼容** | 点与面叠加某些操作无意义 | 检查几何类型组合 |
| **属性字段重名** | 两图层有相同字段名 | 自动重命名冲突字段 |
| **空间范围无交集** | 两图层完全不重叠 | 预检查空间范围 |
| **大数据量性能** | 大图层叠加耗时过长 | 启用空间索引，建议分块 |

### 空间插值参数

| 数据特征 | 推荐方法 | 说明 |
|----------|----------|------|
| 点分布均匀 | IDW（反距离加权） | 计算快速，结果平滑 |
| 精确通过采样点 | Spline（样条函数） | 曲面光滑 |
| 地质、土壤连续变量 | Kriging（克里金） | 考虑空间自相关 |
| 点数量 >1000 | IDW 或自然邻域 | Kriging 大样本效率低 |
| 需要概率估计 | 指示克里金 | 输出概率 |

### 网络分析参数

| 参数隐患 | 问题描述 | 智能处理 |
|----------|----------|----------|
| **网络拓扑错误** | 悬挂节点、伪节点 | 自动检测并调用修复 |
| **起终点不在网络上** | 未正确匹配到路网 | 自动最近点匹配 |
| **单行道/限行** | 忽略通行方向 | 检查路网属性 |
| **权重字段选择** | 距离 vs 时间权重 | 根据目的推荐 |
